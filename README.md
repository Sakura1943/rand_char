# rand_char
Randomly generate fixed-length strings

## Build
```shell
git clone https://github.com/Sakura1943/rand_char.git
cd rand_char
cargo build --release
cd target/release
cp rand_char /usr/bin
```

## Completion
### Bash
```shell
cd rand_char
cd completions
cp rand_char.bash /usr/share/bash-completion/completions/rand_char
```

## Usage
```shell
rand_char
```

## Help
```shell
rand_char --help
```
